Add OAuth2 login with Google and GitHub

Description

Implement OAuth2 social login using Google and GitHub providers. Users should be able to sign in without creating a password. On first login, a new user account should be created and linked to the OAuth identity.

Acceptance criteria

✓User can click "Sign in with Google" and be redirected to Google auth
✓User can click "Sign in with GitHub" and be redirected to GitHub auth
○On first login, a new account is auto-created with email from provider
○Existing accounts with same email are linked automatically
